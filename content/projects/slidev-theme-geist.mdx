# Slidev Theme Geist

Slidev Theme Geist is a theme for [Slidev](https://sli.dev) based on [Vercel's design system](https://vercel.com/design).

## Demo

-   [Live Demo](https://slidev-theme-geist.vercel.app)
-   [Screenshots](#screenshots)

## Install

Add the following frontmatter to your `slides.md`. Start Slidev then it will prompt you to install the theme automatically.

```yml
---
theme: geist
---
```

Learn more about using a theme in the [slidev themes documentation](https://sli.dev/themes/use).
