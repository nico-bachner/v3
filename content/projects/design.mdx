# Design

A design system for my personal projects. It consists of a few different parts:

-   A [Color Scheme](https://github.com/nico-bachner/design/blob/main/packages/colors)
-   [Base Styles](https://github.com/nico-bachner/design/blob/main/packages/css)
-   [Typography](https://github.com/nico-bachner/design/blob/main/packages/typography)
-   [Custom React Hooks](https://github.com/nico-bachner/design/blob/main/packages/hooks)
-   [A Component Library](https://github.com/nico-bachner/design/blob/main/packages/components)
-   [An Icon Library](https://github.com/nico-bachner/design/blob/main/packages/icons)
-   [SVG authoring utilities](https://github.com/nico-bachner/design/blob/main/packages/svg)

For more information, check out the [documentation](https://design.nicobachner.com) or browse the [source code](https://github.com/nico-bachner/design).
