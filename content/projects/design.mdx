# Design

A design system for my personal projects. It consists of a few different parts:

-   [Base Styles](https://github.com/nico-bachner/v3/blob/main/packages/css)
-   A [Color Scheme](https://github.com/nico-bachner/v3/blob/main/packages/colors)
-   A [Typography](https://github.com/nico-bachner/v3/blob/main/packages/typography) System
-   A [Component Library](https://github.com/nico-bachner/v3/blob/main/packages/components-react)
-   An [Icon Library](https://github.com/nico-bachner/v3/blob/main/packages/icons-react)
-   Custom [React Hooks](https://github.com/nico-bachner/v3/blob/main/packages/react-hooks)
-   Custom [SVG Primitives](https://github.com/nico-bachner/v3/blob/main/packages/react-svg)

For more information check out the [overview](/design) or browse the [source code](https://github.com/nico-bachner/v3/blob/main/packages/).
