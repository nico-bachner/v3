# Mac Setup

Here's the procedure I follow when doing a clean (re)install on my Mac.

## System Settings

### Trackpad Settings

The first thing I do when the initial setup procedure is done is customise the trackpad settings:

1.  Turn on _tap to click_
2.  Set _click strength_ to light
3.  Set _tracking speed_ to the maximum value
4.  Turn on _silent clicking_

## Command Line Setup

Install XCode Command Line Tools:

```bash
xcode-select --install
```

### Setup [Homebrew](https://brew.sh)

Install Homebrew:

```bash
curl -fsSL "https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh" | bash
```

Open [taps](https://docs.brew.sh/Taps):

```bash
brew tap homebrew/core
brew tap homebrew/cask
brew tap homebrew/cask-fonts
```

### Install Packages

```bash
brew install starship
brew install fish
brew install node
brew install deno
brew install gh
```

### Install Applications

```bash
brew install --cask visual-studio-code
brew install --cask google-chrome
brew install --cask firefox
brew install --cask discord
brew install --cask signal
```

### Install [Fira Code](https://github.com/tonsky/FiraCode)

```bash
brew install --cask font-fira-code
```

### Setup [Fish Shell](https://fishshell.com)

```bash
echo $(brew --prefix)/bin/fish | sudo tee -a /etc/shells
chsh -s $(brew --prefix)/bin/fish
```

When you open a new terminal window, fish shows a welcome message. To disable this, run the following:

```bash
set fish_greeting ""
```

### Setup [Starship](https://starship.rs)

```bash
echo "starship init fish | source" > ~/.config/fish/config.fish
```

### Setup Projects Directory

```bash
mkdir ~/projects
echo "cd projects" >> ~/.config/fish/config.fish
```
